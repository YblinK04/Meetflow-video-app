🌸 MeetFlow.yy

MeetFlow.yy - это высокопроизводительная SaaS-платформа для мгновенных видеоконференций. Проект ориентирован на максимальную простоту использования (один клик для входа) и премиальный визуальный опыт в стиле "Dark Rose".

🛠 Технологический Стек:

- Framework: Next.js 16 (App Router) — использование самых свежих стандартов React 19.2.
- Styling: Tailwind CSS v4 — конфигурация через CSS-переменные, современный движок PostCSS.
- Video Engine: ZegoCloud SDK — стабильные WebRTC-потоки, поддержка Screen Sharing и групповых звонков.
- Validation: Zod — строгая типизация входных данных.
- Testing: Vitest — Unit-тестирование бизнес-логики и хелперов.
- Icons: Lucide React.
  💡
  Технические особенности (Highlights)
  Проект демонстрирует решение ряда сложных задач фронтенд-разработки:

1. Решение проблемы Hydration Mismatch
   Реализована безопасная генерация уникальных идентификаторов комнат с использованием useEffect. Это позволило избежать рассинхрона между серверным (SSR) и клиентским HTML в React 19.
2. Оптимизация WebRTC соединений (Lobby Pattern)
   Внедрен паттерн "Lobby" (Зал ожидания). Доступ к медиа-устройствам (камера/микрофон) запрашивается только после явного взаимодействия пользователя с интерфейсом. Это решило проблему NotReadableError, возникающую в современных браузерах при попытке автоматического захвата устройств.
3. Отказоустойчивая инициализация
   Для корректной работы в React Strict Mode реализована система «замков» (Initialization Locks) на базе useRef. Это предотвращает дублирование медиа-потоков и конфликты ресурсов при двойном монтировании компонентов.
4. Unit-тестирование
   Бизнес-логика приложения полностью покрыта тестами:
   Валидация Room ID через Zod.
   Алгоритмы генерации криптостойких идентификаторов.

🚀 Как запустить
Клонируйте репозиторий: git clone ...
Установите зависимости: npm install
Создайте .env.local и добавьте ключи:
env
NEXT_PUBLIC_ZEGO_APP_ID=ваш_id
NEXT_PUBLIC_ZEGO_SERVER_SECRET=ваш_secret
Используйте код с осторожностью.
Запустите тесты: npm test
Запустите dev-сервер: npm run dev
